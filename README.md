# 📰 Детекция и классификация пропаганды в новостных текстах
*Отчёт по курсу "Проектные задачи компьютерной лингвистики", весна 2026.* 

*Подгтовили Алимова Полина, Мустакова Кристина, Сидоров Денис*
## Описание и постановка задачи
Проект посвящён задаче автоматического выявления пропаганды в новостных текстах. Работа основана на постановке задачи из SemEval 2020 Task 11. Проект направлен на решение прикладной задачи автоматизации фактчекинга и борьбы с дезинформацией в СМИ.

### Рассматриваются две подзадачи:

- **Span Identification (SI)**
 
Выделение фрагментов текста, содержащих пропаганду.

- **Technique Classification (TC)**
 
Классификация выделенных фрагментов по типам пропагандистских техник.

## Данные
- Используется публичный датасет: **PTC (Propaganda Techniques Corpus)**

Датасет включает:
- текст статьи (один .txt файл на документ)
- два файла разметки
  
Разметка содержит:
- ID статьи
- начальную (включительно) и конечную (не включительно) позиции пропагандистского фрагмента
- (для TC) тип пропагандистской техники

## Архитектура решения
**1. Предобработка данных**

- Извлечение символьных границ пропаганды
- Токенизация текста
- Преобразования символьных спанов в token-level BIO-разметку
- Формирование датасета для обучения моделей

**2. Обучение моделей**
- базовые модели (BiGRU, BERT)
- fine-tuning Qwen2.5-1.5B-Instruct c LoRA

**3. Метрики**
- Стандартные Accuracy и Sent-Accuracy для baseline BiGRU
- Precision, recall, F1-score соответствии с метриками соревнований
